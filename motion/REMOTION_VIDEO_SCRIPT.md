## ANIMATION STYLE NOTES

**Color Palette:**
- Primary: Deep blues (trust, technology)
- Accent: Warm orange/amber (passion, innovation)
- Contrast: Clean whites (clarity)
- Dark mode aesthetic for tech credibility

**Typography:**
- Headlines: Bold, modern sans-serif (like Inter or Space Grotesk)
- Body: Clean, readable
- Code snippets: Monospace (Fira Code or JetBrains Mono)

**Transitions:**
- Smooth fades between acts
- Parallax effects on images
- Subtle particle effects for tech feel
- Ken Burns effect on photos

**Motion:**
- Ease-in-out for smooth professional feel
- Spring animations for energy
- Staggered text reveals for readability
