# Marko StokiÄ‡ Highlight Video

A 15-second Remotion highlight video showcasing Marko's expertise in blockchain and AI.

## Video Structure

1. **Opening (0-3s)**: Name reveal with expertise keywords
2. **Quote Scene 1 (3-6s)**: Speaking photo with CoinDesk quote on AI agent trust
3. **Quote Scene 2 (6-9s)**: Speaking photo with Forbes quote on trusted AI agents
4. **Expertise (9-12s)**: Four key areas with icons
5. **Closing (12-15s)**: Contact information and call to action

## Commands

### Preview the video
```bash
npm start
```

### Render the video
```bash
npm run build
```

Output will be in `out/video.mp4`

### Update Remotion
```bash
npm run upgrade
```

## Customization

- Edit scenes in `src/scenes/`
- Adjust timing in `src/MarkoHighlight.tsx`
- Replace photos in `src/assets/`
- Modify quotes and text directly in the scene files

## Tech Stack

- Remotion 4.0
- React 18
- TypeScript
- Inter font (Google Fonts)
